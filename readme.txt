== Description ==

This plugin includes Custom Field Suits created fields for Yoast SEO Analysis Calculation.

<strong>Read this first:</strong>
This is only for pages cfs fields not the custom post types or other default post types. This filters only applied to the fields created with Custom Field Suits. 


== Installation ==

1. Extract the zip file into your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. On activation it will hide all the theme updates
